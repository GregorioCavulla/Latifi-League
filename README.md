# 🏎️ Latifi League Bolo 2026 Dashboard

Sistema di monitoraggio automatico per la lega F1 Fantasy. 

## 🛠️ Tech Stack
- **Frontend:** HTML5, Tailwind CSS, Chart.js
- **Backend:** Google Sheets (CSV Export)
- **Hosting:** GitHub Pages

## 📊 Come Aggiornare
1. Inserisci i dati nel Google Sheet dedicato.
2. Il sito si aggiorna automaticamente (grazie al sync CSV).
